# خوارزميات | منصة هياكل البيانات التفاعلية

موقع تعليمي تفاعلي يشرح خوارزميات هياكل البيانات بصرياً عبر أنيميشن احترافي
خطوة بخطوة، مع تحليل التعقيد الزمني والمكاني (Big-O). الواجهة عربية بالكامل (RTL)
بوضع داكن عصري.

## المحتوى

- **القوائم المرتبطة**: إضافة (بداية/نهاية/موضع)، حذف (بداية/نهاية)، بحث، وعكس
  القائمة — مع تتبّع المؤشرات وكود زائف متزامن.
- **المكدس (Stack)**: بنية LIFO مع push / pop / peek وأنيميشن للقمة.
- **الطابور (Queue)**: نسختان خطية ودائرية (FIFO) مع توضيح front/rear والالتفاف.
- **خوارزميات الترتيب**: Bubble, Selection, Insertion, Merge, Quick مع تمييز
  المقارنات والتبديلات وعدّادات حيّة.
- **الاستدعاء الذاتي (Recursion)**: جميع الحالات (خطي، ذيلي، شجري، متبادل، متداخل)
  مع عرض مكدس الاستدعاءات ينمو ويتقلّص.
- **الأشجار الثنائية**: BST (إدراج/حذف/بحث + 4 تنقلات)، الكومة (Heapify)، والترتيب
  بالكومة (Heap Sort) مع عرض الشجرة والمصفوفة، وشرح الأشجار الكاملة.
- **التعقيد الزمني**: منحنى نمو تفاعلي يقارن O(1) حتى O(n!) وبطاقات شرح.

## التقنيات

| الغرض | التقنية |
|------|---------|
| الإطار | React 19 + TypeScript + Vite |
| التنسيق | Tailwind CSS v4 |
| الأنيميشن | Framer Motion |
| التوجيه | React Router |
| الرسوم البيانية | Recharts |
| الحالة | Zustand (جاهزة للأقسام القادمة) |

## التشغيل

```bash
npm install
npm run dev      # خادم التطوير على http://localhost:5173
npm run build    # بناء للإنتاج
npm run preview  # معاينة بناء الإنتاج
```
